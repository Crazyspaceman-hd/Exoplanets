Summary: Exoplanet analysis


The purpose of this project was to analyze data about currently discovered exoplanets. To accomplish this I acquired data from the NASA Exoplanet Archive using their site interface.

After downloading the produced CSV some data cleaning was still required to make header names more user friendly and remove columns superflous to the ananlysis. The final data frame contained 2092 rows and 19 columns. An additional data set was formed by removing duplicate entries systems containing multiple planets. This left a list with a single entry for each host star allowing for more convienient analysis of the star itself.

After assembling these data sets, I used Matplotlib to plot various information about the exoplanets and the stars they orbit. The information plotted includes year of discovery, name of observatory where the discovery was made, and total number of planets in system as well as several others. This site provides the cleaned source data and visualizations I created as part of my analysis, as well as descriptions and explanations of the visualizations.
